### Layer 4: Demurrage Engine

#### A. Progressive Demurrage Formula

```javascript
// Calculated on TOTAL balance (Pocket + Vault)

function calculateDemurrage(totalBalance) {
  if (totalBalance <= 5000) {
    // Fireproof base
    return 0
  }
  
  let loss = 0
  
  // Tier 1: 5,000-10,000 (-5%/month)
  if (totalBalance > 5000) {
    const tier1 = Math.min(totalBalance - 5000, 5000)
    loss += tier1 * 0.05
  }
  
  // Tier 2: 10,000-20,000 (-10%/month)
  if (totalBalance > 10000) {
    const tier2 = Math.min(totalBalance - 10000, 10000)
    loss += tier2 * 0.10
  }
  
  // Tier 3: 20,000+ (-15%/month)
  if (totalBalance > 20000) {
    const tier3 = totalBalance - 20000
    loss += tier3 * 0.15
  }
  
  return loss
}

// Applied monthly, calculated daily
function applyDailyDemurrage(balance) {
  const monthlyLoss = calculateDemurrage(balance)
  const dailyLoss = monthlyLoss / 30
  return balance - dailyLoss
}
```

## XI. Philosophical Questions Answered

### Q: "Why give everyone money for free?"

```
A: It's not "free" â€” it's recognition of personhood.

In capitalism:
  - Only labor = value
  - If you can't work â†’ you're worthless
  - Elderly, disabled, caregivers = invisible
  
In Beat:
  - Existence = value
  - You contribute just by being part of community
  - Money = tool for exchange, not whip for control
  
10 BEAT/day is not "enough to live"
It's "enough to participate with dignity"
```

### Q: "Won't people just sit at home and collect BEAT?"

```
A: No, for three reasons:

1. 10 BEAT/day is minimal
   - ~300/month base income
   - In most places = not enough to live alone
   - Need to offer services/goods for more
   
2. Demurrage prevents hoarding
   - Above 5,000 â†’ melts fast
   - Must circulate to get value
   
3. Psychological need
   - Humans want purpose
   - Want recognition
   - Want reputation
   - Base income = freedom to choose work you love
     NOT freedom to do nothing
```

### Q: "What stops inflation?"

```
A: No global prices = no global inflation

Fiat inflation:
  - Central bank prints money
  - Same money supply, same goods
  - Prices rise everywhere
  
Beat:
  - Fixed emission (10/day per person)
  - Local pricing (no global market)
  - Demurrage (money disappears)
  
If too much BEAT in one community:
  - Locals raise prices
  - Or people spend elsewhere
  - Self-regulating
  
Different communities will have different "price levels"
And that's OK (no arbitrage possible)
```

### Q: "Why not just use Bitcoin?"

```
A: Bitcoin optimizes for different goals

Bitcoin:
  - Store of value (digital gold)
  - Scarcity (21M cap)
  - HODL culture (hold forever)
  - Mining = wealth concentration
  - Energy catastrophe
  
Beat:
  - Medium of exchange (digital blood)
  - Abundance (everyone generates)
  - SPEND culture (circulate always)
  - Breathing = equal participation
  - Minimal energy
  
Bitcoin is savings account.
Beat is circulatory system.

Both have value. Different purposes.
```
